Take the access control of Awesome Support.

== Description ==

Awesome Support - Access Manager is a very easy-to-use plugin for Awesome Support.

Take the access control of you Awesome Support install with the possibility to grant or revoke access to any specific user or role.

In addition, if you have [Easy Digital Downloads](https://wordpress.org/plugins/easy-digital-downloads/) running Awesome Support - Access Manager will add a new setting to automatically grant access to users after purchase a product.

= Features of Awesome Support - Access Manager =

* Grant/Revoke access to any specific user (from the user profile).
* Grant/Revoke access to any specific role (from Tickets -> Settings -> Advanced tab).

This plugin requires [Awesome Support](https://wordpress.org/plugins/awesome-support/).

== Installation ==

= From WordPress backend =

1. Navigate to Plugins -> Add new.
2. Click the button "Upload Plugin" next to "Add plugins" title.
3. Upload the downloaded zip file and activate it.

= Direct upload =

1. Upload the downloaded zip file into your `wp-content/plugins/` folder.
2. Unzip the uploaded zip file.
3. Navigate to Plugins menu on your WordPress admin area.
4. Activate this plugin.
